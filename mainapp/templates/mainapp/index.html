A PROPOS

Cette interface web a été conçu pour fournir un interface à l'application de présence automatique.

Elle comprend:

  • Un système de connexion (Login)
  • Une page pour enregistrer un nouveau utilisateur
  • Une page pour lister le registrer de présence

HARDWARE

Les composants électroniques sont la carte Arduino et le lecteur RFID MFRC522.

BACK-END

Le côté back-end comprend le framework backend Django, codé en langage Python.

FRONT-END

Les technologies front-end utilisés sont HTML/CSS, Django et le framework Bootstrap.
